# Task 4: Alert Prioritisation

The process of deciding which to take is called Alert Prioritisation, and it is crucial to ensure timely detection of a threat, especially with many alerts in the queue. Start with critical alerts, then high, medium, and finally low.
Start with the oldest alerts and end with the newest ones.
